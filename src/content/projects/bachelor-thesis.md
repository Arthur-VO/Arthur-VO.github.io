My bachelor's thesis focuses on the security of AI agents within the financial sector. With the rise of the Model Context Protocol (MCP), LLMs are gaining increasing access to external tools and databases. 

This project investigates and automates the detection of security vulnerabilities in these specific integrations. The goal is to prevent malicious actors from gaining access to sensitive financial data or systems via context manipulation and integration exploits.

**Official Title:** Beveiliging van AI‑agents in Fintech: automatische detectie van beveiligingslekken in integraties op basis van het Model Context Protocol (MCP)
